Flexgrid

A Simple SCSS grid based on FlexBox

Fluid

Set column width or let the grid figure it out@include fg-col($size: .25); or @include fg-col();

Responsive

Set different layouts for different screen sizes@include breakpoint(lg) { ... }

Offsets

Add spacing before your columns@include offset(.25)

Auto Widths

Add any number of columns and they resize themselves

Nesting

Rows can be nested inside columns

Horizontal spacing

Start: @include fg-row($horz: start);

End: @include fg-row($horz: end);

Between: @include fg-row($horz: between);

Around: @include fg-row($horz: around);

Vertical spacing

Start: @include fg-row($vert: start);

Center: @include fg-row($vert: center);

End: @include fg-row($vert: end);

Single column alignment

Start: @include fg-col($align: start);

Center: @include fg-col($align: center);

End: @include fg-col($align: end);
